Проект по созданию копии **печально** известной игры Cube World.

## Поддерживаемые платформы

* Windows

## Структура проекта

* Движок (C++17 migration to 20, CMake, Conan)
* Реадактор
* Игра (later...)

## Используемые проекты

* GLM
* GLEW
* GLFW
* boost
* ImGui
* Catch2
* assimp
* stb_image
* json (nlohmann)
